Area 51 Financial

Area 51 Financial is a machine learning group project completed during the WashU Data Analytics program. It utilizes a deep learning neural net model that allows the user to input answers to 9 questions in order to apply for pre-approval for a personal loan. You can visit Area 51 Financial for more information about the project or check out our code at our GitHub Repo. The website is hosted via Heroku so it may a few seconds to initialize and open the webpage.

Bacteria Bio-diversity

This was a solo project completed during the WashU Data Analytics program. This project used a dataset of bacterial samples collected from the belly buttons of around 650 people. The purpose of this project was to create a dashboard to help visualize the types and amount of bacteria for each colleted sample, as well as provide other basic information about the individual it was collected from. The the Python and JS tools that we used for this project were sqlalchemy and flask to access the SQLite database, plotly for the visualizations, and Heroku to deploy the dashboard. You can visit the Bacteria Bio-diversity Dashboard to see what was created as well as the GitHub Repo for the code that was used. The website is hosted via Heroku so it may a few seconds to initialize and open the webpage.
